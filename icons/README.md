# Icons

OS and tool brand icons (128×128, .ico format) for use in Windows Terminal
profiles, VS Code workspace icons, etc. All from iconfinder.com.

## Naming scheme

`<brand>.ico` for the primary, `<brand>-2.ico` for duplicates (different
artistic versions of the same brand from different iconfinder entries).

| File | Brand |
|------|-------|
| `aws.ico` / `aws-2.ico` | Amazon Web Services |
| `azure.ico` | Microsoft Azure |
| `debian.ico` / `debian-2.ico` | Debian |
| `fedora.ico` / `fedora-2.ico` | Fedora |
| `git.ico` | Git |
| `redhat.ico` | Red Hat |
| `ubuntu.ico` | Ubuntu |
| `windows.ico` | Windows |
